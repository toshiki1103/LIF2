# ①課題名 
リフティング記録アプリ

## ②課題内容（どんな作品か） 
選手がリフティングを練習した日付と回数を記録できる

## ③アプリのデプロイURL デプロイしている場合はURLを記入（任意） 

## ④アプリのログイン用IDまたはPassword（ある場合） 
なし

## ⑤工夫した点・こだわった点・難しかった点
①localStorageは文字列しか保存できないため、オブジェクトと文字列の行き来にJSONを使ったところ
②講義の時はキーと値はそれぞれユーザーが入力していたが、キーを固定して複数の記録をまとめて１つのキーで管理できるので、取得する際にgetItem("キー")で全記録が取れて便利だと知ったこと


## ⑥次回トライしたいこと（又は機能）
- リフティング記録入力されたデータを元にグラフ化、最高値なら記録更新！と出すなどUX強化

- 画面上表記している順位はテキストですが、選手間比較して順位が出せるようになると実際に使えるものになりそうです

- 本当はリフティング記録アプリではなく、サッカーの盤面を表現した戦術ボードが理論的にはできることは分かりましたが、難しすぎて断念したので次回以降でトライしたいです
